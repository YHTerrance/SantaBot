# Santa Bot

Santa bot is a farcaster bot that encourages engagement of users on casts.

## Getting Started (Locally)

1. Change directory into `server/`

```bash
cd server/
```

2. Setup Environment Variables

```bash
cp .env.local.example .env.local
```

3. Install packages

```bash
yarn install
```

4. Start dev server

```bash
yarn dev
```
